# OpenWrt 相关编译环境

<br>

+ etkxx-lite-dev

依赖项：uclibcxx

说明：

ETK++ 开发库裁剪版，该安装包提供的运行时库（.so）所包含接口为目前 LBKit 库最小所需。

授权协议：MIT

<br>

+ etkxx-dev

依赖项：libstdcpp

说明：

ETK++ 开发库，该安装包提供的运行时库（.so）包含所有接口（无图形接口硬件实现）。

授权协议：MIT

<br>

+ lite-beapi-dev

依赖项：无（仅考虑安装，但其依赖于 etkxx-dev 或者 etkxx-lite-dev）

说明：

Lite BeAPI 头文件。目前所兼容各处均采用宏或 inline 编码，无运行时库（.so）。

授权协议：MIT
